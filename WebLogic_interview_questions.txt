1. How many ways weblogic domain can be created ?
   3 ways
   1.GUI mode by invoking "config.sh" from weblogic home
   2. BY using WLST.
   3. By using "pack" and "unpack" scripts from WebLogic home
2. Syntax for creating weblogic domain by using pack and unpack commands ?
  Answer:
         To create weblogic domain by using pack and unpack first we have to invoke pack.sh and 
unpack.sh from WL_HOME
pack.sh syntax:
   WL_HOME/common/bin/pack.sh -domain=domain -template=template -template_name=“template_name”[-template_author=“author”][-template_desc=“description”]
[-managed={true|false}
unpack.sh syntax:
WL_HOME/common/bin/unpack.sh -template=wlscltemp.jar -domain=domains/wlscldom -log=wlscldomunpack.log -log_priority=INFO

5. What are diffrent modes of communication in cluster ?
  The communication in  WebLogic cluster is in two type
    1. Unicast mode
    2. multicast mode
 What is Unicast ?
   In unicast communication mode , one of the cluster member acts like lead server which recives the heart beats from remaing
  isntances to know their status.
   It is prefarable for large kind of networks.
 What is multicast ?
   IP multicast is a broadcast technology that enables multiple applications to subscribe to an IP address and port number
   and listen for messages.
   Multicast Ip:224.0.0.0 – 239.255.255.255.
    It is preferable for small kind of networks.

7. what is ssl ? main steps in setting up ssl in weblogic ? how can you know the expire date of existing ssl ?
   SSL:
   Secure Sockets Layer (SSL) provides secure connections by allowing two applications connecting over a network connection
 to authenticate the other's identity and by encrypting the data exchanged between the applications.
  Steps to set up ssl in weblogic:
    1. Obtain an identity and trust for WebLogic server.
    2. Store the private keys, digital certificates, and trusted CA certificates.
    3. Configure the identity and trust keystores for weblogic server in Weblogic admin console.
    4. 4.Set SSL attributes for the private key alias and password in the WebLogic Server Administration Console
  Expire date of ssl:
   It can be verifed on browser, By clicking on lock symol on https url.
